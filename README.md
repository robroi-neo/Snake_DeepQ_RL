## Developer Notes

### Environment Setup (Conda)

**Activate the environment**
```bash
conda activate pygame_env
```

**to generate conda environment**
```bash
conda env export > snake_rl_env.yml
```
**to install env**
```bash
conda env create -f snake_rl_env.yml
```
